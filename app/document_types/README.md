# Catálogo Interno de Tipos Documentais

Este pacote reúne o catálogo interno usado para organizar futuros fluxos de criação de minutas e atas.

Na V5.0, o catálogo é somente uma camada preparatória:

- não gera documentos;
- não chama IA;
- não salva formulários no banco;
- não substitui a revisão humana.

O objetivo é esconder a complexidade técnica do usuário e expor apenas grupos e tipos amigáveis na interface.

Dados essenciais para criação futura de minutas devem ser preenchidos manualmente. OCR poderá ser apoio em etapa posterior, mas não será fonte automática definitiva para nomes, documentos, endereços, valores ou outros dados sensíveis.
